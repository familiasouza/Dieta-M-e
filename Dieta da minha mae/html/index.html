Dieta para abaixar nivel de colesterol alto:

Uma boa dieta pra abaixar o nivel de colesterol no sistema veja uma lista do que pode ocorrer caso o colesterol fique muito alto

  • Presão alta(Hipertensão)
  • Acidente Vascular Cerebral (Avc)
  • Doença Coronária
  • Diabetes tipo 2

Por isso veja uma rotina de dieta completa pra ser seguida todos os dias

Dieta Montada
